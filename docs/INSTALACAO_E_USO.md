# Thermix Studio - Instalacao e Uso

## Requisitos
- Windows 10/11 x64
- Permissao de escrita na pasta de execucao

## Conteudo da publicacao
- ThermixStudio.App.exe
- pasta LatoFont/
- pasta docs/ (este material)

## Como executar
1. Extraia o arquivo .zip para uma pasta local.
2. Execute ThermixStudio.App.exe.
3. Na primeira execucao, a aplicacao criara automaticamente o banco local em thermixStudioDB/.

## Observacoes
- Esta publicacao e self-contained (nao depende de runtime .NET instalado no sistema).
- Se o SmartScreen do Windows exibir alerta, valide o arquivo e execute conforme a politica da empresa.

## Solucao de problemas
- Aplicacao nao abre: confirme que o arquivo foi extraido completamente do .zip.
- Banco nao criado: verifique permissao de escrita na pasta de destino.
- Erro de bloqueio de arquivo: feche instancias anteriores da aplicacao e tente novamente.
